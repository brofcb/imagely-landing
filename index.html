FAQ

Quick answers before you install.

What does Imagely do?

Imagely scans listing pages, helps you filter and select images, and exports selected files or ZIP packages with less friction.

Who is it for?

It’s built for eBay sellers, Shopify sellers, resellers, ecommerce teams, and product researchers who process listings regularly.

Does it work on listing pages?

Yes. Imagely is designed for listing-page workflows. Some websites may require site permissions so images can be read correctly.

Can I export as one ZIP file?

Yes. You can export selected images as a single ZIP file for easier file management and transfer.

How do I install it?

Click any “Install on Chrome” button on this page to open the Chrome Web Store listing and add the extension.

Is it easy to use?

Yes. The interface is task-focused and includes a guided tutorial to help new users start quickly.

Any settings recommended for multi-file downloads?

For smoother multi-file downloads, disable Chrome's “Ask where to save each file before downloading” setting, or use ZIP mode.
